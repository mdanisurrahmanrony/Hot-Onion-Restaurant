# Hot Onion Restaurant
This is a food ordering restaurant app.

## Features
1. Login logout functionality
2. hosted in firebase
3. Add food items to cart
4. Add delivery info
5. Payment the ordered money

### Technology used
1. React
2. Context Api
3. Firebase
4. Stripe Api
5. Boostrap

## npm Command:

1. npm install
2. npm audit fix
3. npm start
4. npm run build
